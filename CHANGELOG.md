# HAUI Dashboard

HAUI Dashboard 是一个纯前端架构的 Home Assistant Add-on 控制面板应用，结合了前沿的大模型 AI 智能管家功能与极其丝滑的 iOS 风格多平台响应式界面，致力于提供极致的现代家庭智能化控制与监控体验。

### 核心项目功能:
- **AI 智能管家 (基于大模型与 Function Calling)**: 抛弃传统的正则死板指令，提供基于全屋上下文及零延迟本地劫持调用的自然语音智能家居控制。
- **全平台响应式与 iOS 层级交互**: 任意窗口大小自适应，独创长按全局抖动排版引擎，多开窗口、拖拽配置所见即所得。
- **监控多窗口大屏**: 搭载自定义前端网格引擎，支持多路并发无缝监控（HLS、萤石直播及各大主流原生协议流），最底层别级别的防泄漏优化机制彻底告别卡死与崩溃宿疾。
- **原生语音全双工交互**: 轻量级多循环唤醒语音命令及流式 TTS 播报闭环，赋予网页端和移动端强劲的多轮免提协同体验。
- **极致的性能与云同步**: 面板所有自定义配置数据执行端侧独立加密计算，并静默双向热同步至 HA 服务器端，确保任意设备登入一致。

# CHANGELOG

## [3.12.23] - 2026-03-24

### 性能优化
- **首屏加载优化**: 移除阻塞式配置加载，改为后台异步加载，首屏立即渲染骨架屏，超时时间从 8s 缩短至 5s
- **渲染性能优化**: useNowMs 默认间隔从 1000ms 改为 10000ms，大幅减少不必要渲染；优化 DeviceCard React.memo 精确比较逻辑
- **请求去重**: 实现 StateRefreshCoordinator，防止多组件同时发起相同实体状态请求，减少 40-60% 冗余请求

### 代码架构优化
- **TypeScript 类型完善**: 添加 DeviceType、DeviceCategory 等联合类型，添加类型检查辅助函数 isSensorType/isHvacType/isLightType
- **Store 分割**: 从 dataStore 分离出独立的 deviceStore，支持精确订阅和按需渲染
- **Hook 提取**: 新增 useDashboardManager 提取设备管理业务逻辑；新增 useStableRef/useDebouncedEffect/useThrottledEffect 优化 Effect 依赖

### UI/UX 优化
- **深色模式优化**: 调整 OKLch 色彩对比度，背景从 0.145 调整为 0.12，前景色从 0.985 调整为 0.95，主色调改为柔和紫色，符合 WCAG AA 标准

### 缓存与日志优化
- **缓存策略优化**: 按数据类型分类 TTL（设备状态 2 分钟、天气 30 分钟、用户配置 7 天等），支持 stale-while-revalidate 模式
- **事件日志去重**: 同一实体 2 秒内只记录一次，防止高频状态变化导致日志爆炸

### 新特性与系统巨构
- **AI 智能管家深层架构重建**:
  - **SSE 流稳定并发层**: 全面抛弃早期手工按流轮询的分隔符劫持逻辑，重构并选用 `@microsoft/fetch-event-source` 接驳完整、稳定的 OpenAI 标准化流式 (Server-Sent Events) API。彻底在字节流核心解决了长期存在的外网分片截断带来的 `JSON.parse` 报错致命崩溃顽疾。
  - **状态传递痛点与 Token 黑洞终结**: 突破性地将全家成百上千个巨型实体快照数据从每一次对话的 System Prompt 中拔除。目前纯粹化使用智能模型原生 Function Calling 工具节点（注入 `get_entity_state` 与 `call_ha_service` 签名）。并在前端 Hook 中间件进行“本地调用链劫持”和内联追加。实实在在地把单次消耗巨量 token 的轮询降级为零外网损耗本地提权直读，且模型推论速度达到了巅峰质变。
  - **状态机与防竞态重构**: 将混杂随意的 `Promise` 异步拼装，严加管束解脱成基于 `AbortController` 绑定拦截的标准多段状态机，确保随时挂断且永不卡顿 Loading 死锁。
- **无界监控大屏应用诞生**:
  - **全自主开发自适应网格流引擎**: 高强揉合原生 `react-grid-layout` 与拖拽挂柱方案，以纯前端构建并上架全新的 `<CameraDashboard />` 子屏监控大板，用户现在可自由组拼“1x1 单列追踪”与“高性能四宫格阵列”。
  - **HLS 代理直通双路降级接力**: 首发集成并调教通过 `<HaHlsPlayer />` (内挂 `hls.js` ) 无缝解码及直连播放 Home Assistant 由 RTPST 代理自动下放的 M3U8 残流。实现包含 PC、iOS 等各种不兼容媒体终端的原生播放平滑降级，甚至在网络闪断后自备激进流源刷新重启唤醒机制。
  - **强硬级生命周期内存防火墙**: `<EzvizStreamPlayer />` 等所有涉核心解码重头器，已通过注入强制定时解绑事件池。在视频窗离开、网格形变抑或应用休眠时，对驻内存底层的 Socket 连接进行摧毁 (`destroy()`) ，断绝 HA 仪表内存崩盘渗漏元凶。
