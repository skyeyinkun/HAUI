## 简介

专为 Home Assistant 打造的现代化高性能 React 控制面板，包含以下特性：

- ⚡ **极其流畅的实时设备控制体验** - WebSocket 实时同步，响应速度 < 100ms
- 🤖 **AI 智能管家** - 支持语音交互和自然语言控制
- 🌤️ **集成精美的动态天气信息** - 自动定位，实时天气数据
- 📷 **原生支持摄像头串流与多画面监控** - HLS、萤石协议支持
- 📊 **详细的设备运行日志与历史数据追踪**
- 🔒 **企业级安全** - 公网访问 PIN 码保护，CSP 安全策略
- 📱 **全平台适配** - 手机、平板、桌面完美适配

## 安装说明

### 方式一：GitHub 仓库安装（推荐）

1. 打开您的 Home Assistant
2. 转到 **设置** → **加载项** → **加载项商店**
3. 点击右上角的三个点菜单 `⋮`，选择 **仓库**
4. 添加仓库地址：`https://github.com/skyeyinkun/HAUI`
5. 刷新页面，找到 **HAUI - 智能家庭中枢** 并点击安装
6. 安装完成后，点击 **启动**
7. 在侧边栏点击 **HAUI** 即可使用

### 方式二：本地开发安装

如果您是开发者且需要本地开发与测试：

1. 在本地运行 `npm run build` 生成 `dist` 文件夹
2. 将整个 `addon` 目录（包含 `dist`）复制到 Home Assistant 宿主机的 `/addons/local/haui_dashboard` 中
3. 在商店中点击右上角的"检查更新"，系统会识别出本地加载项

## 配置指南

### 初始配置

1. 首次启动后，点击右下角 **设置** 按钮
2. 在 **Home Assistant 配置** 标签页中填写：
   - **本地地址**: 您的 HA 内网地址，如 `http://homeassistant.local:8123`
   - **公网地址**（可选）: 外网访问地址
   - **访问令牌**: 从 HA 用户资料页面创建长期令牌
3. 点击 **保存配置**

### AI 配置（可选）

如需使用 AI 语音助手功能：

1. 进入 **设置** → **AI 配置**
2. 选择 AI 提供商（SiliconFlow、OpenAI 等）
3. 填写 API 密钥
4. 选择模型（DeepSeek-V3、GPT-4 等）

### 天气配置

1. 进入 **设置** → **天气配置**
2. 选择所在省份、城市、区县
3. 系统自动获取坐标信息

## 功能特性

### 设备控制

- **开关控制**: 点击设备卡片即可开关
- **亮度调节**: 支持 0-100% 滑动调节
- **色温调节**: 支持冷暖色温调整
- **窗帘控制**: 支持开合度百分比调节
- **场景模式**: 一键激活多设备联动

### AI 智能管家

- **语音交互**: 点击 AI 图标启动语音对话
- **自然语言控制**: "打开客厅灯"等口语化指令
- **设备状态查询**: "卧室温度多少"

### 环境监控

- **传感器数据**: 温湿度、光照实时显示
- **能耗统计**: 用电量统计与趋势分析
- **天气信息**: 实时天气与预报
- **视频监控**: 多路并发视频流

## 安全功能

### 公网访问保护

当通过公网地址访问时，以下高危操作需要 PIN 码确认：

- 门锁控制
- 安防系统解除
- 车库门控制
- 报警器控制

**PIN 码设置**: 首次执行高危操作时自动提示设置

### 数据安全

- 访问令牌本地加密存储
- PIN 码加密存储在浏览器本地
- 所有敏感数据不上传云端
- CSP 内容安全策略防护

---

## 网络细节

- **Ingress 支持**: 完美适配。推荐直接通过 Home Assistant 左侧边栏访问，安全且免密
- **独立端口**: 默认 8099（可通过 Ingress 直接访问，无需额外端口映射）
- **WebSocket**: 使用原生 WebSocket 与 HA 实时通信
- **自动重连**: 连接断开后自动重连，最大延迟 30 秒

## 故障排除

### 无法连接到 Home Assistant

1. 检查 HA 地址是否正确
2. 确认访问令牌是否有效
3. 检查网络连接
4. 查看浏览器控制台网络请求

### 设备状态不同步

1. 检查设备映射配置
2. 刷新页面重新连接
3. 检查 HA 实体状态是否正常

### AI 语音无响应

1. 检查浏览器麦克风权限
2. 确认 AI API 密钥配置正确
3. 检查网络连接
