# HAUI Dashboard 加载项

## 简介
专为 Home Assistant 打造的现代化高性能 React 控制面板，包含以下特性：
- ⚡ 极其流畅的实时设备控制体验
- 🌤️ 集成精美的动态天气信息
- 📷 原生支持摄像头串流与多画面监控
- 📊 详细的设备运行日志与历史数据追踪

## 安装说明
本加载项现已支持从 GitHub 仓库一键安装，无需手动复制文件：
1. 打开您的 Home Assistant。
2. 转到 **配置** -> **加载项**，点击右下角的 **加载项商店**。
3. 点击右上角的三个点菜单 `⋮`，选择 **仓库**。
4. 将本项目的 GitHub 链接添加进去。
5. 刷新页面，在底部找到 **HAUI Dashboard** 并点击安装。
6. 安装完成后，启用“在侧边栏显示”和“Ingress”，点击**启动**即可。

如果您是开发者且需要本地开发与测试：
1. 在本地运行 `npm run build` 生成 `dist` 文件夹。
2. 将整个 `addon` 目录（包含 `dist`）复制到 Home Assistant 宿主机的 `/addons/local/haui_dashboard` 中。
3. 在商店中点击右上角的“检查更新”，系统会识别出本地加载项。

## 配置指南
本控制面板开箱即用，运行在一个轻量化的静态网站服务器容器内，**不需要额外配置任何参数**。
由于完美支持了 Ingress 侧边栏融合技术，它可以免登录直接读取您的 Home Assistant 通信凭证，直接在侧边栏为您呈现极致的管控体验。

## 网络细节
- **Ingress 支持**: 完美适配。推荐直接通过 Home Assistant 左侧边栏访问，安全且免密。
- **独立端口 (Port)**: 默认 80（如果您不通过侧边栏，也可以将它映射出独立端口供局域网直接访问）。
